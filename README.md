## About
The Brand Hub is a private tool created to ensure visual and tonal consistency across all Ambientha communication channels. It works as a single source of truth for the marketing team — accessible on both desktop and mobile for quick references during day-to-day work.

---

## Features
- Style Guide with brand and complementary color palettes (click to copy hex)
- Logo download in SVG and PNG, in black and white versions, across four formats (horizontal, vertical, retraction and stamp)
- Tone of Voice section with communication pillars, words to use and avoid, and ready-to-use response templates
- Mobile-first, fully responsive layout
- Filters for color and format selection on logo downloads
